# ATRI-LK-WebUI

ATRI-LK 项目的 Web 管理界面，基于 Vue 3 + Vite + Element Plus 构建的现代化单页应用。

## ✨ 特性

- 🚀 **Vue 3 Composition API** - 使用最新的 Vue 3 组合式 API
- ⚡ **Vite** - 极速开发体验，热模块替换
- 🎨 **Element Plus** - 丰富的 UI 组件库
- 📦 **TypeScript** - 完整的类型支持
- 🎯 **Pinia** - Vue 3 官方状态管理
- 🔃 **Vue Router** - 单页应用路由
- 📊 **ECharts** - 数据可视化图表
- 🌐 **Axios** - HTTP 请求客户端
- 🎨 **Tailwind CSS** - 实用优先的 CSS 框架

## 🛠️ 技术栈

### 前端框架

- **Vue 3.5.29** - 渐进式 JavaScript 框架
- **Vite 7.3.1** - 下一代前端构建工具
- **TypeScript 5.9.3** - JavaScript 的超集

### UI 与样式

- **Element Plus 2.13.2** - Vue 3 组件库
- **Tailwind CSS 4.2.1** - 实用类优先的 CSS 框架
- **@element-plus/icons-vue** - Element Plus 图标库

### 状态管理与路由

- **Pinia 3.0.4** - Vue 3 官方状态管理库
- **Pinia Plugin Persistedstate 4.7.1** - Pinia 持久化插件
- **Vue Router 5.0.3** - Vue.js 官方路由

### 网络请求

- **Axios 1.13.5** - 基于 Promise 的 HTTP 客户端
- **QS 6.15.0** - 查询字符串解析库

### 数据可视化

- **ECharts 6.0.0** - 强大的数据可视化图表库

### 开发工具

- **ESLint** - 代码质量检查
- **Oxlint** - 高性能 TypeScript/JavaScript linter
- **Prettier** - 代码格式化
- **Vue TSC** - TypeScript 类型检查

## 🚀 快速开始

### 环境要求

- Node.js: ^20.19.0 || >=22.12.0
- npm 或 yarn

### 安装依赖

```bash
npm install
```

### 开发模式

启动开发服务器，支持热重载：

```bash
npm run dev
```

访问 `http://localhost:5173` (端口可能不同，请查看终端输出)

### 生产构建

类型检查、编译并压缩生产版本：

```bash
npm run build
```
